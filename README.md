# 📚 Sistema de Gestión de Biblioteca Escolar (Backend)

Proyecto backend desarrollado con Node.js, Express y MySQL.

---

## 🚀 Tecnologías
- Node.js
- Express
- MySQL
- CORS

---

## 🔐 Funcionalidades

### Login
- Validación de usuario con email y password

### Registro
- Creación de usuarios
- Validación de duplicados

### Libros
- Crear libros
- Listar libros
